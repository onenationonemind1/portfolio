프로젝트 개요

KETI(한국전자기술연구원)에서 수행한 무선전력송신 시스템의 프로젝트입니다. 사용자 요구대로 케이블이나 전선 없이 기기를 충전하는 목표를 달성하였습니다.

  • 개발 기간 : 5주(약 1달)
  • 개발 인원 : 2명
  • 프로젝트 업무
    • 고객 사양에 따른 펌웨어 및 초기/최종 제품 테스트
    • 원인 분석 및 디버깅
    • 회로도, 조립 도면, 테스트 문서 해석
    • 하드웨어 초정밀 납땜 및 케이블 제작



개발 환경

  • AMR: STM32F4 Series, Cortex M4 32bit
  • IDE: STM32CUBE IDE
  • 유지 및 보수: git
  • Language: C, Python



이 프로젝트에서 기여한 부분

  • SPI/I2C/UART 통신을 활용하여 하드웨어 제어
  • GPIO를 활용하여, 최대 속도 2ms의 SPI 드라이브 구현
  • 기재된 15개의 명령어를 모두 성공적으로 구현
  • 고객이 제시한 사양서 중 구현시 발생하는 오류 발견 및 테스트 케이스 및 체크리스트 작성
